# 🧪 SnipIT: Core Intelligence Verification Guide

Use this guide to verify that the "Achievement Gap" has been closed and the proposal features are fully integrated.

## 1. 📖 The Dev-Dex (Searchable Handbook)
*   **Step**: Navigate to the **Dex Tab** (2nd icon).
*   **Action**: Tap on the **"GIT"** or **"LINUX"** cards.
*   **Action**: Use the top search bar to type `pull` or `chmod`.
*   **Check**:
    *   [ ] Does the list update instantly?
    *   [ ] Is the data coming from `handbook.json`?
    *   [ ] Is the command text colored (Syntax Highlighting)?

## 2. 🌐 IP Address Tracker
*   **Step**: In the Dex Tab, tap the **"IP"** segment/tab at the top.
*   **Check**:
    *   [ ] Does it show your **Local IP** (e.g., 192.168.x.x)?
    *   [ ] Does the **Public IP** load after a few seconds (fetching from `api.ipify.org`)?

## 3. 🧩 Regex Pattern Hub
*   **Step**: In the Dex Tab, tap the **"Regex Hub"** card.
*   **Action**: Tap a pattern (like Email Validation) to open the dialog.
*   **Action**: Tap **"Snip IT"** to pre-fill a new snippet.
*   **Check**:
    *   [ ] Does it correctly transition to the New Snip screen with the Regex pre-filled?

## 🎨 4. Syntax Highlighting Engine
*   **Step**: Go to the **Vault** or **New Snip** editor.
*   **Action**: Type or view the following code:
    ```java
    public class SnipIT {
        // This is a comment
        String text = "Hello World";
    }
    ```
*   **Check**:
    *   [ ] Are `public`, `class`, `String` colored **Terminal Green**?
    *   [ ] Is the comment colored **Grey**?
    *   [ ] Is the string "Hello World" colored **Cyan**?

## 📡 5. Beam Optimization (GZIP)
*   **Step**: Find a long snippet (or paste a large chunk of code).
*   **Action**: Tap **"Beam IT"** -> **"QR Mode"**.
*   **Check**:
    *   [ ] Is the QR code generated without a "Too Long" error?
    *   [ ] Does the QR code look moderately dense but scannable? (This confirms GZIP is working).
